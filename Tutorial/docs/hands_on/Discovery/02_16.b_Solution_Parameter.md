## Run-template

Open your local copy of the example run-template of the [onboarding-brewery-solution](https://github.com/Cosmo-Tech/onboarding-brewery-solution/) repository:
[code/run_templates/Example/run.json](https://github.com/Cosmo-Tech/onboarding-brewery-solution/blob/BuildMyApp/Simulator/code/run_templates/Example/run.json)

## New content

To add a new run-step, replace its contents with:
``` json title="run.json"
{
  "steps": [
    {
      "id": "DisplayEnv",
      "command": "python code/run_templates/scripts/display_csm_env.py",
      "useSystemEnvironment": true
    },
    {
      "id": "csm_orc_version",
      "command": "csm-orc",
      "arguments": [
        "--version"
      ],
      "precedents": [
        "DisplayEnv"
      ]
    }
  ]
}
```

## Updating

Run again the updating command:

```bash
babylon apply project/
```

## Checking

Verify that the csm-orc version information appears now in the run logs.
