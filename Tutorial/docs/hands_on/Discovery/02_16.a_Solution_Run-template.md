Here, we will make a basic modification of a run-template.

## Run-template

Open your local copy of the example run-template of the [brewery_sample_solution](https://github.com/Cosmo-Tech/brewery_sample_solution) repository:
[code/run_templates/standalone/run.json](https://github.com/Cosmo-Tech/brewery_sample_solution/blob/main/code/run_templates/standalone/run.json)

## New content

To add a new run-step, replace its contents with:
``` json title="run.json"
{
    "steps": [
        {
            "id": "generate_input_data",
            "command": "python",
            "arguments": [
                "code/run_templates/standalone/generate.py"
            ] 
        }, 
        {
            "id": "engine",
            "command": "csm-simulator",
            "arguments": [
              "-i",
              "BreweryDemoSimulationWithConnector"
            ],
            "precedents": [
              "generate_input_data"
            ],
            "useSystemEnvironment": true
        },
        {
            "id": "csm_orc_version",
            "command": "csm-orc",
            "arguments": [
              "--version"
            ],
            "precedents": [
              "engine"
            ]
        }
    ]
}
```
